# Program menghitung bilangan faktorial

Program ini dibuat untuk menghitung faktorial dari suatu bilangan bulat positif yang dimasukkan oleh pengguna.  
Selain menampilkan output hasil akhir, program juga menampilkan output proses perkalian faktorial.  
Program ini menggunakan kelas "BigInteger" agar dapat menghitung faktorial dengan bilangan yang sangat besar tanpa overflow.
